Solution

Problem

Solution

Concentrate precision-specific tensor upload, weight lookup, dispatch, and download inside one internal module, then let each offload method express only its GELO operation shape.

Benefits

  • leverage: one dispatch pattern
  • tests cover precision matrix
  • delete mirrored branches
